Управление тарифами АТС
Простое Windows Forms приложение на C#, предназначенное для управления автотелефонными станциями (АТС) и их тарифами.

Особенности
Управление АТС: Добавление, редактирование и удаление АТС.
Управление тарифами: Для каждой АТС можно добавлять обычные и льготные тарифы.
Типы тарифов:
Обычный: Базовая цена без скидки.
Льготный: Цена с возможностью указать процент скидки.
Стратегия расчёта стоимости: Реализована с использованием паттерна "Стратегия".
Редактирование тарифов: Возможность изменять базовую цену и скидку в таблице.
Валидация ввода: Проверка корректности вводимых данных (например, числа вместо текста, допустимый диапазон скидки).
Хранение данных: Данные АТС и тарифов сохраняются в JSON-файле.
Импорт/Экспорт JSON: Возможность выгрузить текущие данные в JSON и загрузить их обратно.
Требования
.NET Framework или .NET (в зависимости от вашей целевой платформы).
System.Text.Json для работы с JSON.
Используемые технологии
C#
Windows Forms
JSON (System.Text.Json)
Паттерн "Стратегия"
Структура проекта
ATC.cs: Класс, представляющий АТС.
Tariff.cs: Класс тарифа с поддержкой стратегии расчёта.
CostStrategy.cs: Интерфейс и реализации стратегий расчёта стоимости.
MainForm.cs: Главная форма приложения (список АТС, кнопки управления).
ATCForm.cs: Форма редактирования АТС и её тарифов.
